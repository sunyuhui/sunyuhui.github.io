读了6年大学，依然不知道自己要做什么。

昨天面试了一个妹子，妹子的情况是这样的，大学学的是英语，学霸型，学的很苦（原话），原本打算出国，之后大四保研就一直读下去了，毕业后也做了一段时间的翻译工作。

问她：为什么要转行做前端呢？

按理来说，我学了6年英语，应该去做英语相关的工作，不过男朋友是做 Java 的，所以改行做互联网。

问：那也应该去做 Java 呀，为什么要做前端呢？

因为前端做的东西，我能立马看到效果（这个原因我估计是很多人做前端的原因吧）

又说了句：如果男朋友是做其他行业，那我也会跟着做其他行业的。

这句话算是把我惊到了。

老实说，和妹子聊天其实很愉快，她足够真诚，甚至有点可爱。
